# 贡献指南

欢迎来自社区的 PR！

## 快速开始

```bash
git clone https://github.com/zpf000zpf/15code-verify.git
cd verify
```

## 我们最欢迎的贡献

1. **新探针**：针对新模型 / 新对抗手段
2. **新 Provider 适配**：Gemini / Mistral / DeepSeek 等
3. **Tokenizer 准确度提升**：Claude SentencePiece 对齐
4. **UI/UX**：让报告更清晰、更好分享
5. **文档 / 翻译**：让更多人受益

## 不接受的 PR

- 任何会让工具**失去中立性**的改动（如特定供应商的豁免逻辑）
- 把扫描默认**强行开启榜单贡献**
- 在报告中使用"造假 / 欺诈"等主观措辞

## 法务红线

本项目坚持 [LEADERBOARD_POLICY.md](LEADERBOARD_POLICY.md) 的中立原则。
任何改动若影响法律安全性，必须经 legal@15code.com 审核。
