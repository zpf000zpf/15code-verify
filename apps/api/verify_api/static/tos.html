服务条款

版本 v1.1 · 生效日期 2026-04-23

使用 15code Verify（以下简称"本工具"）即视为同意本条款。

1. 工具性质

本工具是 15code 提供的免费技术测试工具，用于帮助用户评估第三方 LLM API 服务的质量与一致性。

2. 用户担保

使用本工具扫描任何 API 端点时，您担保：(a) 您拥有该 API key 的合法使用权； (b) 您有权对该 endpoint 执行技术测试；(c) 您的使用不违反第三方服务商的服务协议； (d) 您已理解并接受第 3 条关于数据公开的默认行为。

3. 数据处理与公开行为

API Key：一次性扫描仅内存驻留，扫描结束立即销毁，不落盘。

扫描内容：只使用预置探针题库，不读取或存储用户私人 prompt；扫描产生的 response 仅作指纹对账用途，24 小时内自动删除原文。

榜单公开（默认）：自 v1.1 起，所有扫描的匿名聚合结果默认进入公开榜单，供应商通过扫描目标 base_url 的 host 名标识。发布内容仅限：

  • 聚合统计指标（均值、置信区间、样本量）
  • 探针输出的分类得分（真伪度、计费偏差、缓存、QoS）
发布内容绝不包含：API key、prompt 原文、response 原文、用户 IP、个人标识。

如您不希望某次扫描结果进入榜单，请使用 API 调用并设置 publish_to_leaderboard: false，或通过 dispute@15code.com 申请删除。

4. 免责声明

本工具按"现状"提供。15code 不对 检测结果的绝对准确性、结果适用于特定商业决策、工具 24/7 可用性作出任何担保。

检测报告不构成对任何服务商的法律评价、商品质量结论或投资/交易建议。

责任上限：¥0（本工具完全免费，零对价）。

5. 禁止滥用

禁止将本工具用于：对未授权系统的扫描；恶意制造虚假数据污染榜单；任何违反适用法律的行为。

6. 联系方式

一般咨询 support@15code.com · 争议申诉 dispute@15code.com
