榜单政策

版本 v1.1 · 生效日期 2026-04-23

核心立场

15code Verify 是技术工具 + 数据聚合平台，不是评级机构。本榜单：

  • 不对任何服务商作出商业评价或法律判断
  • 呈现所有扫描的匿名聚合统计结果（v1.1 起默认公开，详见服务条款第 3 条）
  • 所有数据均基于公开可复现的方法论

默认公开政策（v1.1 起）

自 2026-04-23 起，本工具采用默认公开模式：

  • Web 表单提交的扫描 → 默认进入榜单聚合
  • API 调用如希望私有，需显式传 publish_to_leaderboard: false
  • 用户在 Web 表单提交前会看到醒目的公开告知文字
  • 供应商可通过 dispute@15code.com 申请下线或申辩

展示规则

  • 单一供应商少于 100 个扫描样本不公开展示
  • 所有指标以 均值 ± 置信区间 展示
  • 永远不展示单次扫描记录
  • 使用中立技术语言（"与 X 一致性 Y%"），不使用"造假/欺诈"等主观措辞

供应商权益

申辩权：每个榜单条目设"供应商回复"入口，与用户数据并列展示。

复查申诉：邮箱 dispute@15code.com，48h 响应。

退出机制：供应商可申请不公开，条目标"应要求不展示"。

方法论透明

打分算法、探针设计原理完全开源，版本化发布。详见 GitHub。

免责声明

本榜单数据反映特定时间点的技术指标观测，不构成：对任何服务商商业行为的法律评价；对任何商品质量的官方结论；交易或采购决策建议。
