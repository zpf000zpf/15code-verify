# Probe Bank

This directory hosts **public, illustrative probes only**. The real probe bank
used by 15code production systems is private and rotated frequently.

- `*.public.yaml`  — public sample probes (safe to disclose)
- `*.private.yaml` — NOT tracked (gitignored), only on 15code infra

The public set exists for:
- External researchers to reproduce methodology
- Self-hosted deployments (baseline functionality)
- Educational transparency
