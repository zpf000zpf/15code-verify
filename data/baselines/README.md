# Ground Truth Baselines

Stores per-model baseline statistics collected by direct calls to official
APIs (Anthropic / OpenAI / Gemini / etc).

Files:
  `baselines-YYYYMMDD.json`  — weekly snapshots

**IMPORTANT**: These are collected using 15code's OWN official keys,
never user-provided keys. See docs/SECURITY.md.

Not tracked in git (each snapshot can be large).
